# rx-samsara

一个`react`应用的数据流管理框架<br>
它是在**响应式编程**`js`库 `RxJS 6` 的基础上开发的，并且在使用时也需要依赖于`rxjs`的`observable`；<br>
在`rx-samsara`的世界里，`observable`是基本单元(数据类型)，就像 `function`之于`javascript`。<br>

## 安装

通过 npm

```
$ npm install rx-samsara --save
```
