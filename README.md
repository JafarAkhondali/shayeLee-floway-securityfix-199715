# Floway

**`Floway`是`flow-way`的简写，意为数据流的流动轨道！**<br>
她是`RxJS v6`的扩展库，主要功能是作为`React`应用的状态(数据流)管理工具。

## 特点

- 提供了 `state` 操作符来创建和定义状态数据
- 提供了 `subscription` 装饰器使React视图组件可以响应状态的变化
- 使用 `dispatch action` 的模式来更新状态
- 使用`RxJS`的操作符来动态计算状态(类似计算属性)
- 多个组件可以共享状态数据
- 可以在任意位置修改任意组件的状态
